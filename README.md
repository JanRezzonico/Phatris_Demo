# Phatris
## A Tetris clone built with the Phaser 3 Engine
<b> Authors: </b> Jan Rezzonico & Luca Lorenzon

### Getting started

Create `.env` containing the connection string to the DB `MONGODB_URI=mongodb+srv://...` and `PORT=...`

```bash
npm i
npm run start
```
